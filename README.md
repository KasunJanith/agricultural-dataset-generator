# Agricultural Translation Dataset Generator

A full-stack web application for generating agricultural domain-specific translation datasets from Sinhala to English.

## Features

- Generate random agricultural terms and sentences in Sinhala
- Automatic Singlish and English translation variants
- 10 agricultural subdomains
- Duplicate prevention
- CSV export functionality
- Real-time statistics
